# E1-2 미션 피드백

## 기능 요구사항

### 1. 메뉴 표시 및 주요 기능 동작

> 프로그램 실행 시 메뉴가 정상적으로 표시되고, 퀴즈 풀기/추가/목록/점수 확인 기능이 모두 동작하는가?

- **결과:** Pass
- **사유:** 프로젝트 기능 목록을 통해 퀴즈 풀기, 추가, 목록, 점수 확인 기능이 제공됨을 명확히 제시하였습니다.

### 2. 정답/오답 판정 및 입력 오류 처리

> 퀴즈 풀기 시 정답/오답 판정이 정확하고, 최소 입력 오류 케이스에 대한 처리가 되어 있는가?

- **결과:** Fail
- **사유:** 제출된 결과물에서 공백, 문자, 범위 밖 입력 등 입력 오류 케이스 처리 내용을 확인할 수 없습니다.

### 3. 데이터 영속성

> 프로그램 종료 후 재실행해도 추가한 퀴즈와 최고 점수가 유지되는가?

- **결과:** Pass
- **사유:** 프로그램을 종료해도 퀴즈 데이터와 점수가 state.json 파일에 자동 저장되어 유지된다고 명확히 명시하였습니다.

### 4. 기본 퀴즈 포함 여부

> 기본 퀴즈가 5개 이상 포함되어 있는가?

- **결과:** Pass
- **사유:** 데이터 파일 설명에서 첫 실행 시 기본 퀴즈 5개로 자동 생성된다는 사실을 명시하였습니다.
